Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select an element in the DOM using JavaScript. As like ID: getElementById("id"), Class: getElementsByClassName("class"), Tag: getElementsByTagName("tag"), Name: getElementsByName("name").

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML Returns or sets the HTML inside an element, including tags.
InnerText Returns only visible text (ignores hidden elements).It Doesn't include HTML tags.
TextContent Returns all text inside an element, including hidden ones. More performance-efficient than innerText.

Question-3: What is event delegation in the DOM?

Event Delegation in the DOM is a JavaScript technique where you attach a single event listener to a parent element instead of adding multiple listeners to individual child element.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a process that propagates events up through the DOM tree, starting with the innermost element
